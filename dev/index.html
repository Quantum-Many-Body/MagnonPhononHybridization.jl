MagnonPhononHybridization

Magnon-phonon hybridized systems.

Introduction

Magnon-phonon hybridization in magnetically ordered quantum lattice systems is based on the QuantumLattices pack, the TightBindingApproximation pack, and the SpinWaveTheory pack.

Installation

In Julia v1.8+, please type ] in the REPL to use the package mode, then type this command:

pkg> add MagnonPhononHybridization
